# Switch 开关选择器

---

## 如何使用

:::demo

```html
<Text>默认样式</Text>
<Switch checked/>
<Switch/>
<Text>推荐展示样式</Text>
<Switch checked/>
<Switch/>
```

:::


## Switch 参数

|  H5   | 属性    | 类型    | 默认值 | 说明                           |
| --- | ------- | ------- | ------ | ------------------------------ |
| √  | checked | Boolean | false  | 是否选中                       |
| √   | type    | String  | switch | 样式，有效值：switch, checkbox |
|     | color   | Color   |        | switch 的颜色，同 css 的 color |
